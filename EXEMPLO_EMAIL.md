## 💡 Dicas para Melhorar seu Email

### ✅ Faça:
- ✓ Seja profissional mas autêntico
- ✓ Cite tecnologias específicas que a vaga pede
- ✓ Demonstre conhecimento da empresa
- ✓ Seja conciso (3-5 parágrafos)
- ✓ Use formatação clara
- ✓ Revise erros de digitação
- ✓ Inclua seu contato

### ❌ Evite:
- ✗ Emails muito longos (mais de 1 página)
- ✗ Linguagem muito informal
- ✗ Explicar TUDO seu currículo (isso está no PDF)
- ✗ Reclamações sobre empregos anteriores
- ✗ Erros de português/digitação
- ✗ Termos muito técnicos sem contexto

## 🎯 Dica Extra: Emails Personalizados

Se quiser enviar emails mais PERSONALIZADOS:

**Opção 1**: Enviar poucos emails com personalização maior
```
Prezados [NOME DA EMPRESA],

Vi que vocês estão em busca de um Desenvolvedor React...
[resto do email personalizado]
```

**Opção 2**: Usar a ferramenta para um template genérico
```
Prezados Recrutadores,

Sou desenvolvedor com experiência em [tecnologias]...
[email geral que funciona para todos]
```

Com esses templates você pode esperar:
- **2-5%** de taxa de resposta (é normal!)
- **10-15%** de emails que abrem a mensagem
- **1-3%** que chamam para entrevista

Quanto mais emails enviar, mais chances de sucesso!

## 📧 Exemplo de Lista de Emails

Para usar na aplicação, copie a lista assim:

```
rh@empresa1.com.br; contato@empresa2.com; jobs@empresa3.com.br; recrutamento@empresa4.com
```

**Dicas para encontrar emails:**
1. LinkedIn - contate recrutadores da empresa
2. Site da empresa - procure página de carreiras
3. Plataformas: LinkedIn Jobs, Glassdoor, Indeed
4. Email genérico: careers@, jobs@, hr@, contato@
