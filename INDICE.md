## 🎓 APRENDIZADOS-CHAVE

### Tecnologias Usadas:
- **React** - Frontend (interface bonita)
- **Node.js** - Runtime JavaScript
- **Netlify Functions** - Backend serverless
- **Nodemailer** - Envio de emails
- **Gmail SMTP** - Servidor de emails

### Segurança:
- Senha de App (não a senha normal)
- HTTPS (criptografado)
- Sem armazenamento de dados
- Cada sessão é nova

### Performance:
- 10 segundos entre emails (estável)
- ~500 emails por hora máximo
- Frontend responsivo
- Sem travamentos

## 💡 DICAS OURO

1. **Use SENHA DE APP**, não a senha normal do Gmail
2. **Teste a conexão** antes de enviar
3. **Mantenha a aba aberta** durante todo o envio
4. **Envie em lotes** de 50-100 emails
5. **Personalize quando possível** (aumenta resposta)
6. **Não envie spam** (prejudica reputação)

## 🚨 CUIDADOS IMPORTANTES
