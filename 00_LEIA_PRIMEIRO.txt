Bem-vindo! Este arquivo explica o que você tem nas mãos
e por onde começar.

  O QUE É ESSE PROJETO?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Uma aplicação web completa para enviar emails em massa com
PDF anexado, e ver quem abriu cada email.

  ✓ Você preenche os emails, escreve a mensagem, anexa o PDF
  ✓ A aplicação envia um por um com delay de 10 segundos
  ✓ Uma aba de Rastreamento mostra quem abriu e quando

Custo: R$ 0. Tudo roda em serviços gratuitos.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ANTES DE FAZER O DEPLOY — 4 COISAS PARA PREPARAR
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. GMAIL — gere uma senha de app (não a senha normal):
     https://myaccount.google.com/apppasswords
     Você precisa de autenticação em dois fatores ativada.

  2. GITHUB — crie um repositório para hospedar o código:
     https://github.com/new

  3. NETLIFY — onde a aplicação vai rodar (grátis):
     https://app.netlify.com

  4. SUPABASE — para o painel de rastreamento (grátis):
     https://supabase.com
     Após criar o projeto, execute SUPABASE_SETUP.sql no
     SQL Editor do painel.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  VARIÁVEIS DE AMBIENTE (configurar no Netlify)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Em Site settings → Environment variables, adicione:

  GMAIL_USER            → seu-email@gmail.com
  GMAIL_APP_PASSWORD    → senha de 16 caracteres do Gmail
  SUPABASE_URL          → URL do projeto no Supabase
  SUPABASE_SERVICE_KEY  → service_role key do Supabase

  Sem essas variáveis, a aplicação não consegue enviar emails.
